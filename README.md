# Metasploitable 2 Lab

## Objective

My goal for this lab is to better understand an attacker's process, how attackers can exploit common vulnerabilities, chain vulnerabilities together, and access systems that have not been properly hardened. With this practice I aim to be better equipped to defend against attacks, while identifying opportunities for improved security. I am working my way through the vulnerabilities, and will update my findings as I go along!

### Skills Learned
- Network & Host enumeration  
- Strengthened familiarity with Metasploit  
- Improved Nmap skills  
- OSINT vulnerability identification 

### Tools Used
- Metasploitable2 - Intentionally vulnerable Ubuntu machine
- Kali Linux VM
- Metasploit
- Nmap
